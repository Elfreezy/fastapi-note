### Описание

Реализован функционал приложения-ежедневника. Присутствует возможность создания, редактирования, удаления (CRUD), архивирования записей и вывода всех строк в табличный формат.

**Stack**: Python, FastAPI, DOCKER, PostgreSQL

### Обзор
Основная страница выводит перечень всех хранимых строк с полями: название, описание, архив. Доступны действия добавления новой строки, редактирование и удаление существующих (столбец - Action). Архивные записи отмечаются соотв. иконкой.

### Локальный запуск 

Предполагается наличие docker'а на компьютере. Запуск контейнеров осуществляется через docker-compose

В директории проекта выполнить:

```
docker-compose up
```

### Примечания

Пример настройки файла .env для локального запуска:

```
POSTGRES_USER=postgres
POSTGRES_PASSWORD=secret
POSTGRES_URI=postgresql+psycopg2://<username>:<userpwd>@<iphost>:<port>/<dbname>
```

Пример POSTGRES_URI:

```
POSTGRES_URI=postgresql+psycopg2://postgres:secret@172.18.0.2:5432/demo
```
